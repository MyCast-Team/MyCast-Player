Type Nom de la classe Auteur Date Description
CLASS MainFrameController Vincent Rossignol et Thomas Fouan 01/02/2016 This is the main class that manage our application. We load the different components and plugins.
CLASS MediacaseController Vincent Rossignol 14/06/2016 This is the controller of the mediacase component. The mediacase is the list of the user's medias. You can search in this list and drag and drop medias into the playlist
CLASS MenuBarController Thomas Fouan 10/03/2016 This is the controller to manage the MenuBar. It defines functions to call on events.
CLASS PlaylistController Vincent Rossignol 25/04/2016 This is the controller of the playlist component. The playlist is a list of medias that will be played in order. You can delete medias or reset the playlist. The playlist is saved in a file between two MyShare use.
CLASS PluginController Pierre Lochouarn 30/05/2016 This is the controller for the plugin panel. You can manage multiple things like search, download or uninstall plugins.
CLASS StatusBarController Thomas Fouan 18/06/2016 This controller manage the little status bar at the bottom of the application.
CLASS SuggestionController Pierre Lochouarn 30/05/2016 This is the controller for the suggestion panel. The application will give to the user some ideas about movies to watch and music to listen.
